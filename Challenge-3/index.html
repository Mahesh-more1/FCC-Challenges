Introduction

Python is an interpreted, high-level, general-purpose programming language. Created by Guido van Rossum and first released in 1991, Python's design philosophy emphasizes code readability with its notable use of significant whitespace.

Python features include:

  • Simple, easy to learn syntax
  • Interpreted language (no compilation needed)
  • Dynamically typed
  • Supports multiple programming paradigms (procedural, object-oriented, functional)
  • Comprehensive standard library

Python is used for:

  • Web development (Django, Flask)
  • Data analysis and machine learning (Pandas, NumPy, TensorFlow)
  • Automation and scripting
  • Scientific computing
  • Desktop applications

This guide assumes you have the following basic background:

  • A general understanding of programming concepts
  • Basic computer operation skills
  • Some experience with any programming language is helpful but not required
Python vs Other Languages

Python differs from languages like Java and C++ in several key ways:

  • Python is dynamically typed (no variable type declarations)
  • Uses indentation for block structure rather than curly braces
  • Generally more concise than Java or C++
  • Interpreted rather than compiled
  • Extensive standard library included

Compared to JavaScript:

  • Python is strongly typed (no implicit type conversion)
  • Python is primarily server-side while JavaScript runs in browsers
  • Python has more consistent behavior across implementations
Hello World

To get started with Python, here's the simplest program:

print("Hello, World!")

To run this program:

  1. Save it in a file with .py extension (e.g., hello.py)
  2. Run it from command line: python hello.py

Or you can use the Python interactive interpreter by typing python in your terminal.

Variables

Variables in Python are created when you first assign a value to them:

x = 5 y = "Hello" z = 3.14

Variable names:

  • Can contain letters, numbers and underscores
  • Cannot start with a number
  • Are case-sensitive
  • Should be descriptive (e.g., user_name rather than un)

Python has no command for declaring a variable. A variable is created the moment you first assign a value to it.

Data Types

Python has the following built-in data types:

  • Text: str
  • Numeric: int, float, complex
  • Sequence: list, tuple, range
  • Mapping: dict
  • Set: set, frozenset
  • Boolean: bool
  • Binary: bytes, bytearray, memoryview

You can check the type of any variable with the type() function:

x = 5 print(type(x)) # Output: <class 'int'>
If...Else Statements

Python uses indentation to define blocks of code:

a = 200 b = 33 if b > a: print("b is greater than a") elif a == b: print("a and b are equal") else: print("a is greater than b")

Python supports the usual logical conditions:

  • Equals: a == b
  • Not equals: a != b
  • Less than: a < b
  • Greater than: a > b
  • Less than or equal to: a <= b
  • Greater than or equal to: a >= b
While Loops

With the while loop we can execute a set of statements as long as a condition is true.

i = 1 while i < 6: print(i) i += 1

Loop control statements:

  • break - stops the loop
  • continue - skips to next iteration
  • else - runs when loop finishes normally
i = 0 while i < 6: i += 1 if i == 3: continue print(i) else: print("i is no longer less than 6")
Functions

A function is a block of code which only runs when it is called. You can pass data, known as parameters, into a function.

def my_function(name): print("Hello " + name) my_function("Alice")

Functions can return values:

def add_numbers(a, b): return a + b result = add_numbers(5, 3) print(result) # Output: 8

Default parameter values:

def greet(name="World"): print(f"Hello, {name}!") greet() # Output: Hello, World! greet("Alice") # Output: Hello, Alice!
Lists

Lists are used to store multiple items in a single variable:

fruits = ["apple", "banana", "cherry"] print(fruits)

List operations:

  • Access items: fruits[0] (first item)
  • Negative indexing: fruits[-1] (last item)
  • Range of indexes: fruits[1:3]
  • Change value: fruits[1] = "blueberry"
  • Add items: fruits.append("orange")
  • Remove items: fruits.remove("banana")

List comprehension:

numbers = [1, 2, 3, 4] squares = [n**2 for n in numbers] print(squares) # Output: [1, 4, 9, 16]
Dictionaries

Dictionaries store data values in key:value pairs:

person = { "name": "Alice", "age": 25, "city": "New York" } print(person["name"]) # Output: Alice

Dictionary operations:

  • Access items: person["age"]
  • Add items: person["email"] = "alice@example.com"
  • Remove items: person.pop("age")
  • Check if key exists: "name" in person
  • Loop through: for key, value in person.items():

Dictionary comprehension:

numbers = [1, 2, 3, 4] squares = {n: n**2 for n in numbers} print(squares) # Output: {1: 1, 2: 4, 3: 9, 4: 16}
